# Withdraw- 💸

A simple and professional withdrawal tracking page built with HTML/CSS/JS.  
This project helps users simulate and manage withdrawal records in a clean interface.

---

## 🚀 Features
- Easy-to-use withdrawal tracking interface
- Responsive design for mobile and desktop
- Simple HTML/CSS/JS structure (no heavy frameworks)
- Ready for GitHub Pages deployment
